# PClippy

![Securethelogs.com](https://ctrla1tdel.files.wordpress.com/2020/02/psclippy.gif)

PSClippy is a POC to show a potential attack point using PowerShell.

<b>To Run</b>
Download the script and edit the following lines:

- Line 40: If you wish to write to file, remove the '#'.
- Line 48: If you wish to tweak the counter before upload (PasteBin).
- Line 53: API key for PasteBin.
- Line 59: Name of your Paste. Make sure it's unique so you can easily search for it.
- Line 79: If you wish to extend the timer past 5 seconds. 

For more information, read my site post here: https://securethelogs.com/2020/02/13/stealing-from-copy-and-paste/


*Please only use this for testing and not for malicious purposes
